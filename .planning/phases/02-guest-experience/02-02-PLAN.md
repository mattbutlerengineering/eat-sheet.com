<task type="auto">
  <name>Task 2: Add table combining API endpoints</name>
  <files>src/server/routes/floor-plans.ts</files>
  <action>
Add two new endpoints to floor-plans route:

1. POST /:floorPlanId/tables/combine
   - Body: { table_ids: string[], label: string }
   - Creates new table with combined_from = join(table_ids)
   - Sets max_capacity = sum of child table max_capacity
   - Hides or marks child tables as unavailable

2. POST /:floorPlanId/tables/:id/uncombine
   - Restores child tables to available status
   - Deletes combined table

3. Update GET /:id/tables to include combined_from in response
   - Also include occupied_since when status is 'occupied' (calculated from reservation seated time)
  </action>
  <verify>
npm run build 2>&1 | head -20
  </verify>
  <done>API supports combining and un-combining tables</done>
</task>
