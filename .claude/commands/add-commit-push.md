---
description: Add, commit, and push changes to git
---

Please add all changes, create a git commit, and push to the remote repository. Follow these steps:

1. Run `git status` to see all changes
2. Run `git add .` to stage all changes
3. Create a descriptive commit message based on the changes made
4. Commit with the message
5. Push to origin/main

Make sure to include the Claude Code footer in the commit message.
